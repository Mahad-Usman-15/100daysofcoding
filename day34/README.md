# Currency Converter  

## Features
- prompt user to give a amount and currency
- can convert usd,pkr,pound,inr to usd,pkr,inr,pound
- if the currency is valid give the converted amount otherwise show a error message

## How to create
- store the values of the currencies in a dict
- ask the user to give amount,from and to currency
- if the pair is in the dict give the user converted amount otherwise return an error

## Tech stack
plain python 3.1

## Success Criteria
- currency is converted into given currency.
